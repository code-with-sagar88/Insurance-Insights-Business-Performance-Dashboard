# 📊 Insurance Insights & Business Performance Dashboard

An interactive Tableau dashboard project analyzing insurance business performance, premiums, claims, and profitability to generate actionable insights for better business decisions.

👤 Author: Sagar Dabhade

---

# 🎯 Business Problem

Insurance companies need to monitor premiums, claims, profitability, and customer trends across cities and insurers.  

This project helps analyze insurance business performance and identify growth opportunities using interactive dashboards.

---

# 📂 Project Structure

- insurance_finance_records.csv → Dataset used for analysis  
- dashboards/ → Tableau dashboard screenshots  

---

# 🚀 Key Highlights

✅ Analysis of Premiums, Claims, and Customer Age Distribution  
✅ City-wise & Insurer-wise Value Creation  
✅ Profitability by Policy Tenure and Cost Ratios  
✅ Stakeholder-wise Assets Under Management  
✅ Shareholding Distribution & Trend Analysis  

---

# 📊 Dataset Description

The dataset includes the following columns:

- Policy ID  
- City  
- Insurer  
- Premium Amount  
- Claim Amount  
- Policy Tenure  
- Customer Age  
- Cost Ratio  
- Stakeholder  
- Shareholding Percentage

# 📌 Key Insights

🔹 Goa generates the highest value among cities (₹128.1B)  
🔹 Bancassurance partners contribute maximum insurer value (₹139.91B)  
🔹 Profitability varies significantly with policy tenure  
🔹 Majority of assets are contributed by People and Distributors  
🔹 FII hold the largest share in ownership (38.4%)
